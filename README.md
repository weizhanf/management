## 三、贯穿全书的两条主线（先建立全局观）

整门课其实围绕两个核心思想展开，理解了它们，80% 的题都有抓手：

1. **贡献式思维（Contribution approach）**——把成本拆成"变动 / 固定"，
   `贡献毛益 = 收入 − 变动成本`，再用它"先盖固定成本、剩下是利润"。
   → 支撑第 **2、3、5、6** 章。

2. **相关性思维（Relevant information）**——决策只看"**未来会发生、且各方案之间有差异**"的成本与收入；
   沉没成本（已花的钱）一律不相关。
   → 支撑第 **5、6、11** 章。

第 **7、9、10** 章是"控制与评价"主线：预算（事前）→ 责任会计（事中/事后）→ 业绩评价（ROI/RI/EVA）。
