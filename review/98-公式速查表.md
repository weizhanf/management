## 第2章 本量利分析（CVP）
```
单位贡献毛益 = 单价 − 单位变动成本
总贡献毛益   = 销售收入 − 变动成本 = 销量 × 单位贡献毛益
贡献毛益率   = 单位贡献毛益 ÷ 单价

营业利润 = 销量 × 单位贡献毛益 − 固定成本

盈亏平衡销量   = 固定成本 ÷ 单位贡献毛益
盈亏平衡销售额 = 固定成本 ÷ 贡献毛益率

目标利润所需销量   = (固定成本 + 目标利润) ÷ 单位贡献毛益
目标利润所需销售额 = (固定成本 + 目标利润) ÷ 贡献毛益率
税前利润 = 税后目标利润 ÷ (1 − 税率)

安全边际   = 预计销售 − 盈亏平衡销售
安全边际率 = 安全边际 ÷ 预计销售
```

## 第3章 成本性态的计量
```
线性成本函数：Y = a + bX   （a=固定成本总额, b=单位变动成本, X=业务量）

高低点法：
  b = (最高业务量点成本 − 最低业务量点成本) ÷ (最高业务量 − 最低业务量)
  a = 某点总成本 − b × 该点业务量
  ※ 高低点按"业务量 X"取，不是按成本取！
```

## 第5章 定价/营销决策
```
主要成本(Prime cost)   = 直接材料 + 直接人工
加工成本(Conversion)   = 直接人工 + 制造费用

特殊订单：增量贡献毛益 = 增量收入 − 增量变动成本（>0 且有闲置产能 → 接）

成本加成定价：价格 = 成本基础 × (1 + 加成率)
            加成率 = (价格 − 成本基础) ÷ 成本基础

最低报价 = 全部变动成本 + 可避免固定成本

目标成本法：目标成本 = 目标售价 − 目标利润   （现有成本 ≤ 目标成本 → 投产）
```

## 第6章 生产经营决策
```
自制 vs 外购：比较【可避免成本】与【外购成本】，取低者；腾出产能计机会成本

设备更新相关性：
  旧设备账面价值/折旧/账面冲销 → 不相关（沉没）
  旧设备处置价值、新设备购置成本、未来经营成本差异 → 相关

联产品：增量收入 > 分离点后可分成本 → 继续加工（联合成本不相关）

约束资源排序 = 单位贡献毛益 ÷ 单位产品耗用的约束资源
```

## 第7章 全面预算
```
采购预算 = 预计销货成本 + 期末存货 − 期初存货
生产预算 = 预计销量 + 期末产成品 − 期初产成品

现金预算：
  期末现金 = 期初现金 + 现金收入 − 现金支出 ± 融资
  ※ 不含折旧；销售按账期比例拆月收现
```

## 第10章 分权组织管理控制
```
ROI = 营业利润 ÷ 投资额
    = (营业利润 ÷ 销售收入) × (销售收入 ÷ 投资额)   [杜邦分解]

剩余收益 RI = 营业利润 − 投资额 × 最低报酬率

EVA = 税后营业利润 − 税后资本成本率 × (总资产 − 流动负债)

最优转移价格 = 单位付现(增量)成本 + 单位机会成本
  · 有闲置产能：机会成本=0 → 下限=单位变动成本
  · 满负荷：    机会成本=对外贡献毛益 → ≈ 市价
```

## 第11章 资本预算
```
NPV = Σ(各年现金流 × 折现系数) − 初始投资        （≥0 接受）
  · 等额现金流 → 用 Table 2 年金系数
  · 不等额     → 逐年用 Table 1 单笔系数

IRR：使 NPV=0 的折现率
  年金情形：年金系数 = 初始投资 ÷ 年现金流，反查 Table 2 得 IRR

回收期 = 初始投资 ÷ 年现金流（均匀时）

ARR = 平均年净利润增加额 ÷ 初始(或平均)投资

含税现金流：
  税后经营现金流 = 税前经营现金流 × (1 − 税率)
  折旧抵税额     = 折旧 × 税率
  税后净现金流   = 上面两项之和
```
