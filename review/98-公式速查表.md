## 第2章 本量利分析（CVP）
```
单位贡献毛益 = 单价 − 单位变动成本
总贡献毛益   = 销售收入 − 变动成本 = 销量 × 单位贡献毛益
贡献毛益率   = 单位贡献毛益 ÷ 单价

营业利润 = 销量 × 单位贡献毛益 − 固定成本

盈亏平衡销量   = 固定成本 ÷ 单位贡献毛益
盈亏平衡销售额 = 固定成本 ÷ 贡献毛益率

目标利润所需销量   = (固定成本 + 目标利润) ÷ 单位贡献毛益
目标利润所需销售额 = (固定成本 + 目标利润) ÷ 贡献毛益率
税前利润 = 税后目标利润 ÷ (1 − 税率)

安全边际   = 预计销售 − 盈亏平衡销售
安全边际率 = 安全边际 ÷ 预计销售
```

## 第3章 成本性态的计量（🔧仅概念，不要求手算）
```
线性成本函数：Y = a + bX   （a=固定成本总额, b=单位变动成本, X=业务量）
约束性 vs 酌量性固定成本（概念高频）

高低点法（仅理解原理，考试不手算）：
  b = (高点成本 − 低点成本) ÷ (高业务量 − 低业务量)；a = 总成本 − b×业务量
```

## 第5章 定价/营销决策
```
主要成本(Prime cost)   = 直接材料 + 直接人工
加工成本(Conversion)   = 直接人工 + 制造费用

特殊订单：增量贡献毛益 = 增量收入 − 增量变动成本（>0 且有闲置产能 → 接）

成本加成定价：价格 = 成本基础 × (1 + 加成率)
            加成率 = (价格 − 成本基础) ÷ 成本基础

最低报价 = 全部变动成本 + 可避免固定成本

目标成本法：目标成本 = 目标售价 − 目标利润   （现有成本 ≤ 目标成本 → 投产）
```

## 第6章 生产经营决策
```
自制 vs 外购：比较【可避免成本】与【外购成本】，取低者；腾出产能计机会成本

设备更新相关性：
  旧设备账面价值/折旧/账面冲销 → 不相关（沉没）
  旧设备处置价值、新设备购置成本、未来经营成本差异 → 相关

联产品：增量收入 > 分离点后可分成本 → 继续加工（联合成本不相关）

约束资源排序 = 单位贡献毛益 ÷ 单位产品耗用的约束资源
```

## 第7章 全面预算（🔧仅概念，考试不编表）
```
（理解即可）全面预算 = 经营预算 + 财务预算；编制起点 = 销售预算
采购预算 = 预计销货成本 + 期末存货 − 期初存货   （供理解，不要求手编）
现金预算：期末现金 = 期初现金 + 现金收入 − 现金支出 ± 融资（不含折旧）
```

## 第11章 资本预算
```
NPV = Σ(各年现金流 × 折现系数) − 初始投资        （≥0 接受）
  · 等额现金流 → 用 Table 2 年金系数
  · 不等额     → 逐年用 Table 1 单笔系数

IRR：使 NPV=0 的折现率
  年金情形：年金系数 = 初始投资 ÷ 年现金流，反查 Table 2 得 IRR

回收期 = 初始投资 ÷ 年现金流（均匀时）

ARR = 平均年净利润增加额 ÷ 投资额（初始 或 平均）
  · 有残值：平均投资额 = (初始投资 + 残值) ÷ 2
  · ARR 用会计净利润(已扣折旧)，非现金流

全项目法/差量法：多方案各算NPV再比 / 只算两方案差额NPV（结论一致）

含税现金流：
  税后经营现金流 = 税前经营现金流 × (1 − 税率)
  折旧抵税额     = 折旧 × 税率
  税后净现金流   = 上面两项之和
```
