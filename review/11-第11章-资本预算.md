# 第11章 · 资本预算

> 英文：Capital Budgeting
> 评价**长期投资项目**是否划算。核心是**货币时间价值**：把未来现金流折算成今天的价值。计算量大、易失分，**必须练熟查表**。

---

## 一、本章在考试中的地位

⭐⭐⭐ **核心**。高频大题：**净现值 NPV**、**内含报酬率 IRR**、含**所得税与折旧抵税**的现金流、**敏感性分析**、回收期。

## 二、核心概念（中英对照）

| 概念 | 含义 |
|------|------|
| 资本预算 Capital budgeting | 长期资产投资的规划与评价 |
| 货币时间价值 | 今天的 1 元 > 未来的 1 元 |
| 折现 Discounting | 把未来现金流换算成现值 |
| 必要报酬率 / 资本成本 / 最低报酬率 | 折现率，项目应达到的门槛(hurdle rate) |
| 现金流量 Cash flow | 资本预算**只看现金流，不看会计利润** |
| 折现系数 Discount factor | 查现值表得到的乘数 |
| 年金 Annuity | 各期金额相等的现金流 |

> **两张现值表**（务必分清）：
> - **Table 1：单笔 $1 的现值** → 用于"某一年发生的一次性现金流"。
> - **Table 2：年金 $1 的现值** → 用于"连续若干年每年相等的现金流"。
> 看错表 = 整题崩，这是第11章头号失分点。

## 三、四种评价方法

| 方法 | 是否折现 | 决策规则 |
|------|:------:|---------|
| 净现值 NPV | ✅ | NPV ≥ 0 接受；多方案选 NPV 最大 |
| 内含报酬率 IRR | ✅ | IRR ≥ 必要报酬率则接受 |
| 回收期 Payback | ❌ | 回收期越短越好（不考虑时间价值与回收后现金流） |
| 会计报酬率 ARR | ❌ | 用会计利润，越高越好 |

### (1) 净现值 NPV
```
NPV = Σ(各年现金流 × 对应折现系数) − 初始投资
```
- 年现金流相等 → 用 **Table 2 年金系数** 一次乘完。
- 年现金流不等 → 逐年用 **Table 1 系数** 折现再求和。

### (2) 内含报酬率 IRR
使 NPV = 0 的折现率。年金情形可反查表：
```
年金现值系数 = 初始投资 ÷ 年现金流
再到 Table 2 对应年数那一行，找出该系数对应的利率，即 IRR
```

### (3) 回收期 Payback
```
回收期（现金流均匀） = 初始投资 ÷ 每年现金流
（现金流不均匀则逐年累加，直到累计 = 初始投资）
```

### (4) 会计报酬率 ARR（老师点名要会手算）
```
ARR = 增加的平均年净利润 ÷ 投资额（初始 或 平均）

若资产有期末残值(Salvage value)，平均投资额 = (初始投资 + 残值) ÷ 2
```
> 注意：ARR 用**会计净利润**（已扣折旧），不是现金流；这是它和 NPV/回收期最大的区别。
> **例**：投资 100,000、无残值、年均净利润 12,000 → ARR(按初始) = 12,000 ÷ 100,000 = 12%；
> 若按平均投资 = (100,000 + 0)/2 = 50,000 → ARR = 12,000 ÷ 50,000 = 24%。**看清题目要哪种口径。**

### 全项目法 vs 差量法（老师点名 p481）
多方案比较时有两种列法，结论一致，二选一即可：
- **全项目法 Total project approach**：把**每个方案各自**的全部现金流都折现，分别求 NPV，再比大小。
- **差量法 Differential approach**：只看**两方案之间的差额**现金流，求差额 NPV。
> 方案多时全项目法更清晰；只比两方案时差量法更快。

## 四、所得税与折旧抵税（务必掌握）

含税时现金流分两部分：

```
① 经营现金流的税后额 = 税前经营现金净流入 × (1 − 税率)
② 折旧抵税额(depreciation tax shield) = 折旧 × 税率
   （折旧本身不是现金流出，但它减少应税利润，从而少交税 → 节省现金）

税后净现金流 = ① + ②
```

### 例题 〔11-41〕税率 40%
- 税前经营现金流入 = 收入 1,200,000 − 付现成本 600,000 = 600,000
- 所得税流出 @40% = 240,000 → **税后经营现金流 = 360,000**
- 折旧 400,000 的抵税 = 400,000 × 40% = **160,000**
- **现金总影响 = 360,000 + 160,000 = 520,000**

> 记牢两句：**折旧不是现金流，但折旧能抵税（×税率）就是省下的现金。** 经营现金流要乘 (1−税率)。

## 五、例题精讲（取自你的练习题）

### 例1 · NPV 与查表 〔11-34〕
NPV = 年现金流 × 年金系数 − 初始投资。表格反解练习：
- 8 年、14%：年金系数 4.6389 → 4.6389×10,000 − 投资 = 5,613 → 投资=**40,776**
- 28 年、25%：系数 3.9923 → 8,000×3.9923 − 25,000 = **6,938**

### 例2 · 设备购置 NPV（相关现金）〔11-35〕
```
初始净支出 = 23,000（以旧换新中，"超额折让"其实是售价调整，账面价值不相关；
              真正相关的是增量付现 23,000）
经营节约现值 = 5,000 × 6.1944(12年12%年金系数) = 30,972
NPV = 30,972 − 23,000 = +7,972 > 0  →  应购买
```

### 例3 · 不等额现金流 + 递延年金 〔11-36〕
把各年净现金流先相减简化，再分段折现：
- 第1–3年净流入各 50,000 → 50,000 × 2.3216(3年年金) = 116,080
- 第4–5年净流入各 150,000（递延3年的2年年金）→ 150,000 × 1.6467 × 0.6750 = 166,728
- 合计现值 282,808 − 初始投资 215,000 = **NPV 67,808**

### 例4 · 由 NPV 求 IRR 〔11-37〕
同一项目在不同折现率下：
- 10%：15,000×3.7908 − 54,072 = +2,790
- 12%：15,000×3.6048 − 54,072 = **0** → **IRR = 12%**
- 14%：15,000×3.4331 − 54,072 = −2,575
> IRR 就是让 NPV=0 的那个利率。本例 IRR(12%) > 资本成本(10%) → 接受；与 NPV>0 的结论一致。

### 例5 · 敏感性分析 〔11-38〕
考察寿命、现金流变动对结果的影响：
- 回收期 = 30,000 ÷ 5,000 = 6 年
- 基准 NPV = 5,000×6.8137 − 30,000 = +4,069
- 若年现金流降到 3,500：NPV = 3,500×6.8137 − 30,000 = −6,152（由正转负 → 对现金流敏感）

## 六、易错点与陷阱

1. **看清用哪张表**：一次性现金流用 Table 1，等额年金用 Table 2。
2. **只认现金流，不认会计利润**；折旧不是现金流出，但**折旧×税率**是抵税现金流入。
3. **沉没成本/账面价值不相关**（与第6章一致，如 11-35 旧设备账面价值）。
4. **初始投资**要用"现金等价成本"（含税费、扣真实折让）。
5. 递延年金：先算"普通年金现值"，再乘"递延期的单笔折现系数"。
6. 现金**流出写负、流入写正**，最后 NPV = 流入现值 − 投资。
7. NPV 与 IRR 一般给出相同的"接受/拒绝"结论；互斥方案排序可能冲突，**优先看 NPV**。
