## 二、核心概念（中英对照）

| 概念 | 含义 |
|------|------|
| 全面预算 Master budget | 涵盖全公司的综合预算，= 经营预算 + 财务预算 |
| 经营预算 Operating budget | 反映经营活动，最终落到**预计利润表** |
| 财务预算 Financial budget | 反映资金与财务状况，含资本预算、现金预算、预计资产负债表 |
| 预计报表 Pro forma statements | 用预算数据编出的预计财务报表 |
| 编制起点 | **销售预算**（一切预算的源头） |

### 预算的作用（简答高频）
1. **强制规划**（promotes planning）
2. **协调与沟通**（coordination & communication）各部门
3. **提供业绩评价基准**（benchmark for performance）
4. **激励员工**（motivation）

> 补充：**参与式预算(participative budgeting)**让下级参与编制，提升认同感，但可能产生**预算松弛(budgetary slack)**（故意低估收入/高估成本以便轻松达标）。

## 三、全面预算的结构（必须会画这张图）

```
全面预算 Master Budget
├── 经营预算 Operating Budget
│   ① 销售预算 Sales budget          ← 起点
│   ② 采购/生产预算 Purchases/Production budget
│   ③ 销货成本预算 Cost of goods sold budget
│   ④ 经营费用预算 Operating expense budget
│   └─⑤ 预计利润表 Budgeted income statement
│
└── 财务预算 Financial Budget
    ⑥ 资本预算 Capital budget（长期资产购置）
    ⑦ 现金预算 Cash budget
    └─⑧ 预计资产负债表 Budgeted balance sheet
```

> 编制顺序：**销售 → 生产/采购 → 各项费用 → 预计利润表 → 现金预算 → 预计资产负债表。** 环环相扣，前一个的输出是后一个的输入。

## 四、解题套路一：采购预算（商业企业）

```
本期采购 = 预计销售成本 + 期末存货 − 期初存货
```
（制造企业对应"生产预算"：本期生产量 = 预计销量 + 期末产成品 − 期初产成品）

---

## 五、解题套路二：现金预算（最常考）

现金预算分四块，按下式逐月推进：

```
   期初现金余额
+  现金收入（本期收现 + 以前赊销在本期收回）
─────────────────────────────────
=  可动用现金合计
−  现金支出（采购付现、工资、费用、所得税、购置设备、利息等）
─────────────────────────────────
=  融资前现金余额
±  融资活动（不足最低额则借款(+)，有余则还款与付息(−)）
─────────────────────────────────
=  期末现金余额（即下期期初）
```

**关键技巧——销售收现的时间分布**：题目常说"当月收 60%，次月收 30%，再次月收 10%"。
要按这个比例把**各月销售额拆到对应月份的现金收入**里。同理，赊购付款也有账期。

> 折旧不是现金支出 → **现金预算里绝不出现折旧**（但预计利润表里有折旧）。

### 微型示例
某月：期初现金 5,000；本月销售收现 40,000；采购付现 28,000；付工资费用 9,000；最低现金需求 4,000。
```
可动用 = 5,000 + 40,000 = 45,000
支出   = 28,000 + 9,000 = 37,000
融资前余额 = 45,000 − 37,000 = 8,000 ≥ 4,000 → 无需借款
期末现金 = 8,000
```

## 六、补充：其他预算方法（概念了解）

| 方法 | 英文 | 要点 |
|------|------|------|
| 作业基础预算 | Activity-based budgeting | 按作业及其成本动因编预算 |
| 零基预算 | Zero-based budgeting | 每期从零开始论证每笔支出，而非在上期基础上增减 |
| 滚动/连续预算 | Rolling/continuous budget | 永远保持向前 12 个月，做一月补一月 |

> 静态预算(static/master budget) 是固定业务量下的预算；**弹性预算(flexible budget)与差异分析在第 8 章**（本次考试不在范围）。

## 七、易错点与陷阱

1. **现金预算不含折旧、不含赊销中尚未收现的部分**——只认"真正收/付的现金"。
2. 销售收现要按账期比例拆月，别把全部销售额当本月现金。
3. 采购预算别忘了**期初/期末存货**的调整。
4. 利润 ≠ 现金：预计利润表有利润，现金预算可能仍缺钱（应收账款占用）。
5. 编制起点永远是**销售预算**。
