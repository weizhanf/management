## 二、高频术语中英对照（按主题分组）

### 成本性态（第2、3章）
| 英文 | 中文 | 一句话理解 |
|------|------|-----------|
| Cost behavior | 成本性态 | 成本随业务量怎么变 |
| Variable cost | 变动成本 | 总额随业务量成正比变化 |
| Fixed cost | 固定成本 | 相关范围内总额不变 |
| Mixed cost | 混合成本 | 一部分固定+一部分变动 |
| Relevant range | 相关范围 | 成本性态假设成立的业务量区间 |
| Cost driver | 成本动因 | 引起成本变化的因素（如机器工时） |
| Contribution margin (CM) | 贡献毛益 / 边际贡献 | 收入 − 变动成本 |
| Contribution margin ratio | 贡献毛益率 | 单位贡献毛益 ÷ 单价 |
| Break-even point | 盈亏平衡点 / 保本点 | 利润为 0 时的销量或销售额 |
| Margin of safety | 安全边际 | 预计销售 − 保本销售 |
| Operating leverage | 经营杠杆 | 固定成本比重越高，杠杆越大 |
| Committed fixed cost | 约束性固定成本 | 难以削减（厂房折旧、高管薪酬） |
| Discretionary fixed cost | 酌量性固定成本 | 管理层可自主增减（广告、研发） |
| High-low method | 高低点法 | 用最高/最低业务量两点估成本函数 |
| Least-squares regression | 最小二乘回归法 | 用全部数据点拟合成本直线 |

### 相关信息与决策（第5、6章）
| 英文 | 中文 | 一句话理解 |
|------|------|-----------|
| Relevant cost | 相关成本 | 未来发生 + 方案间有差异 |
| Sunk cost | 沉没成本 | 已发生、不可改变 → **永远不相关** |
| Opportunity cost | 机会成本 | 放弃的最佳替代方案的收益 |
| Outlay cost | 付现成本 | 需要实际支付现金的成本 |
| Differential / Incremental cost | 差量 / 增量成本 | 两方案之间的成本差额 |
| Avoidable cost | 可避免成本 | 选择某方案后能省下的成本（相关） |
| Unavoidable cost | 不可避免成本 | 无论如何都要发生（不相关） |
| Special order | 特殊订单 | 一次性、低价的额外订单 |
| Make-or-buy | 自制或外购 | 零部件自己造还是外购 |
| Joint products | 联产品 | 同一过程产出的多种主要产品 |
| Split-off point | 分离点 | 联产品可单独辨认的那一点 |
| Separable cost | 可分成本 | 分离点之后单独加工的成本 |
| Contribution approach | 贡献式（法） | 按成本性态编排利润表 |
| Absorption approach | 完全成本（吸收）法 | 按职能（制造/销管）编排利润表 |
| Target costing | 目标成本法 | 目标成本 = 目标售价 − 目标利润 |

### 预算与控制（第7、9、10章）
| 英文 | 中文 | 一句话理解 |
|------|------|-----------|
| Master budget | 全面预算 / 总预算 | 经营预算 + 财务预算的总和 |
| Operating budget | 经营预算 | 销售→采购/生产→费用→预计利润表 |
| Financial budget | 财务预算 | 资本预算+现金预算+预计资产负债表 |
| Cash budget | 现金预算 | 现金收入−支出−融资 |
| Sales budget | 销售预算 | 全部预算的起点 |
| Pro forma / Budgeted statements | 预计财务报表 | 预算编出的报表 |
| Responsibility center | 责任中心 | 成本/收入/利润/投资中心 |
| Cost / Profit / Investment center | 成本/利润/投资中心 | 对应可控范围不同 |
| Goal congruence | 目标一致性 | 个人目标与组织目标一致 |
| Controllability | 可控性 | 该责任人能否影响该项目 |
| Balanced scorecard | 平衡计分卡 | 财务/客户/内部流程/学习成长 四维度 |
| Return on investment (ROI) | 投资报酬率 | 营业利润 ÷ 投资额 |
| Residual income (RI) | 剩余收益 | 营业利润 − 投资额×资本成本率 |
| Economic value added (EVA) | 经济增加值 | 税后营业利润 − 资本成本 |
| Transfer price | 内部转移价格 | 内部部门间交易的计价 |

### 资本预算（第11章）
| 英文 | 中文 | 一句话理解 |
|------|------|-----------|
| Capital budgeting | 资本预算 | 长期投资项目的评价 |
| Net present value (NPV) | 净现值 | 现金流现值 − 初始投资 |
| Internal rate of return (IRR) | 内含报酬率 | 使 NPV = 0 的折现率 |
| Required rate of return / Hurdle rate | 必要报酬率 / 最低报酬率 | 折现用的标准 |
| Cost of capital | 资本成本 | 资金的机会成本 |
| Payback period | 回收期 | 收回初始投资所需年数 |
| Accounting rate of return (ARR) | 会计报酬率 | 平均会计利润 ÷ 投资 |
| Discount factor | 折现系数 | 查现值表得到的系数 |
| Annuity | 年金 | 每期相等的现金流 |
| Depreciation tax shield | 折旧抵税 | 折旧 × 税率 = 节省的税 |
| After-tax cash flow | 税后现金流 | 税前现金流 × (1 − 税率) |
