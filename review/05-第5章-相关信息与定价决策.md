# 第5章 · 相关信息与决策：定价/营销决策

> 英文：Relevant Information for Decision Making, with a Focus on Pricing/Marketing Decisions
> 本章建立"**相关性思维**"，并应用于特殊订单、定价、目标成本。

---

## 一、本章在考试中的地位

⭐⭐⭐ **核心**。高频大题：**特殊订单接/不接**、**目标成本法**、**投标/定价（成本加成）**。

---

## 二、核心概念：什么是"相关信息"

> **相关成本/收入 = ① 未来才发生 + ② 各方案之间有差异。** 两个条件缺一不可。

| 概念 | 含义 | 是否相关 |
|------|------|:------:|
| 沉没成本 Sunk cost | 已经发生、无法改变的成本（如已购设备的账面价值） | ❌ 永不相关 |
| 未来差异成本 | 不同方案下会不同的未来成本 | ✅ 相关 |
| 各方案相同的未来成本 | 无论选哪个都一样 | ❌ 不相关 |
| 机会成本 Opportunity cost | 选此方案而放弃的最佳替代收益 | ✅ 相关 |

**决策流程模型**：信息 → 预测 → 决策模型（比较各方案相关数据）→ 决策 → 评价。
**定性因素**也要考虑（客户关系、质量、声誉、员工士气），有时会压倒微小的数量差异。

---

## 三、两种利润表格式（必须会区分）

| | 贡献式（Contribution approach） | 完全成本式（Absorption approach） |
|---|---|---|
| 分类依据 | 成本**性态**（变动/固定） | 成本**职能**（制造/销管） |
| 中间结果 | 贡献毛益 | 毛利(Gross margin) |
| 用途 | **内部决策、CVP** | 对外财务报告 |

贡献式结构：
```
销售收入
− 变动制造成本 + 变动销管费用 = 变动成本合计
= 贡献毛益
− 固定制造费用 − 固定销管费用
= 营业利润
```

> 补充术语：**主要成本(Prime cost) = 直接材料 + 直接人工**；**加工成本(Conversion cost) = 直接人工 + 制造费用**。（见例题 5-34）

## 四、决策套路一：特殊订单（接 or 不接）

> 关键前提：**有闲置生产能力**时才适用增量分析（不挤占正常销售）。

```
第1步  只看这笔订单带来的【增量收入】与【增量变动成本】
第2步  增量贡献毛益 = 增量收入 − 增量变动成本
第3步  若现有固定成本不变 → 增量贡献毛益 > 0 即应接受
第4步  补定性判断：是否会冲击正常售价/老客户？能否带来长期客户？
```

### 例题 〔5-36〕
正常 2,000 件，新增特殊订单 100 件：

| | 不接订单 | 特殊订单影响 | 接订单 |
|---|---|---|---|
| 销量 | 2,000 | 100 | 2,100 |
| 销售收入 | 36,000 | 1,500 | 37,500 |
| 采购成本 | 20,000 | 1,000 | 21,000 |
| 变动印刷成本 | 4,000 | 200 | 4,200 |
| 变动成本合计 | 24,000 | 1,200 | 25,200 |
| **贡献毛益** | 12,000 | **300** | 12,300 |
| 固定成本 | 8,000 | **0** | 8,000 |
| **营业利润** | 4,000 | **+300** | 4,300 |

**结论**：接受订单利润 +300，**短期应接受**。但若担心冲击正常 $18 售价、引来其他客户压价，损失可能 > 300，则可拒绝（定性）。

## 五、决策套路二：定价

### (1) 经济学视角
价格弹性 + "边际收入 = 边际成本"处利润最大。考试多为概念了解。

### (2) 成本加成定价（Cost-plus）
```
价格 = 成本基础 × (1 + 加成率)
```
成本基础可选：单位变动成本、单位制造成本、单位完全成本……基础越窄，加成率越高。

**加成率计算**〔5-43〕：加成率 =（价格 − 成本基础）÷ 成本基础
- 基础=变动成本36,000：(90,000−36,000)/36,000 = **150%**
- 基础=制造成本50,000：(90,000−50,000)/50,000 = **80%**
- 基础=完全成本30,000：(90,000−30,000)/30,000 = **200%**

### (3) 投标/最低报价〔5-44〕
```
最低报价(只覆盖相关成本) = 全部变动成本 + 可避免固定成本
目标报价 = 全部相关与不可避免成本 + 期望利润加成
```
- 总变动成本 124,000 + 可避免固定 9,000 = **最低报价 133,000**
- 加上不可避免固定 35,000 = 总成本 168,000；加 20% 加成 33,600 → **目标报价 201,600**

## 六、决策套路三：目标成本法（Target costing）

> 思路相反：**先由市场定售价，倒挤出允许的成本。**

```
目标成本 = 目标售价 − 目标利润
若 现有预计单位成本 ≤ 目标成本 → 可投产；否则需降本或放弃
```

### 例题 〔5-46〕
- 目标售价 230，目标利润 = 20%×230 = 46 → **目标成本 = 230 − 46 = 184**
- 现有单位成本 = 总成本 10,000,000 ÷ 50,000 件 = **200** > 184 → **不应投产**
- 若降本：节约 = 40%×5,000,000 − 1,100,000 = 900,000；单位节约 = 900,000÷50,000 = 18
- 新单位成本 = 200 − 18 = **182 < 184 → 可投产** ✓

## 七、易错点与陷阱

1. **沉没成本一律剔除**（如已投入的研发、旧设备账面价值）。
2. 特殊订单分析**只看增量**，不要把原有固定成本摊进去再比较。
3. 接特殊订单的前提是**有闲置产能**；若需挤占正常订单，要计入**机会成本**。
4. 成本加成的加成率随"成本基础"不同而不同，看清题目用哪个基础。
5. 目标成本法是"**价格倒推成本**"，与成本加成方向相反，别搞混。
