# 第6章 · 相关信息与决策：生产经营决策

> 英文：Relevant Information for Decision Making, with a Focus on Operational Decisions
> 把第5章的"相关性思维"用于内部生产决策：**自制/外购、联产品深加工、设备更新、约束资源**。

---

## 一、本章在考试中的地位

⭐⭐⭐ **核心**。高频大题：**自制或外购**、**设备更新（保留 or 替换）**、**联产品分离点后是否深加工**。也常考"沉没成本/账面价值不相关""单位成本会误导"等辨析题。

---

## 二、核心概念回顾

| 概念 | 含义 | 决策中地位 |
|------|------|-----------|
| 沉没成本 Sunk cost | 已发生、不可改变（旧设备账面价值、已花研发） | **永不相关** |
| 机会成本 Opportunity cost | 放弃的最佳替代方案收益（如厂房出租收入、自用收益） | 相关 |
| 可避免成本 Avoidable cost | 选某方案后可省下的成本 | 相关 |
| 不可避免成本 | 无论选哪个都发生（如已分摊的总部费用） | 不相关 |
| 联产品/分离点/可分成本 | 见"决策套路三" | 联合成本不相关 |

> **黄金口诀**：决策只比"**未来 + 有差异**"的现金性成本与收入。账面价值、已分摊的固定费用，统统先划掉。

---

## 三、决策套路一：自制或外购（Make-or-Buy）

```
第1步  分别列出【自制总相关成本】与【外购总相关成本】
       - 自制方：直接材料+直接人工+变动制造费+【可避免】固定制造费
       - 外购方：外购单价×数量
       - 注意：已分摊但不可避免的固定费用（如分摊的厂房折旧）两边都不算
第2步  比较两者，取较低者
第3步  若外购后腾出的产能可另作他用 → 把【机会成本/其他用途贡献】计入
第4步  补定性：供应商可靠性、质量、交期（JIT 尤其看重）
```

### 例题A 〔6-32〕Bose
| | 自制 | 外购 |
|---|---|---|
| 直接材料 | 300,000 | |
| 可避免间接人工 | 30,000 | |
| 可避免物料 | 20,000 | |
| 已分摊占用成本(不可避免) | 0 | |
| 外购成本 | | 330,000 |
| **相关成本合计** | **350,000** | **330,000** |

→ 外购便宜 20,000。但因差额小，**定性因素（供应可靠性、质量、控制力）可能主导**，公司可能愿"投资"这 2 万换取供应保障。注意：经理曾错误地忽略了 50,000 可避免固定成本。

### 例题B 〔6-33 + 6-34〕Nantucket Nectars
- 基本比较：自制相关成本 230,000 < 外购 250,000 → **应自制**（省 20,000）。
- 进阶〔6-34〕考虑腾出产能的用途（机会成本）：

| 方案 | 自制 | 外购+闲置 | 外购+他用 | 外购+出租 |
|---|---|---|---|---|
| 其他活动贡献 | | | +75 | |
| 租金收入 | | | | +55 |
| 瓶子相关成本 | (230) | (250) | (250) | (250) |
| **净相关成本** | **(230)** | (250) | **(175)** | (195) |

→ **外购并把产能用于其他活动**净成本最低（−175），最优。

## 四、决策套路二：设备更新（保留 or 替换）

> 四个关键项目的相关性判断（**背下来**）：

| 项目 | 是否相关 | 说明 |
|------|:------:|------|
| 旧设备**账面价值/折旧/账面冲销** | ❌ 不相关 | 沉没成本 |
| 旧设备**现在的处置(变现)价值** | ✅ 相关 | 未来现金流入 |
| 新设备**购置成本** | ✅ 相关 | 未来现金流出 |
| 未来**经营成本**差异 | ✅ 相关 | 各方案不同 |

### 例题 〔6-40〕（5 年合计法）
| | 保留 | 替换 | 差异 |
|---|---|---|---|
| 现金经营成本 | 22,500 | 10,000 | 12,500 |
| 旧机折旧 / 一次性冲销 | 5,000 | 5,000 | 0（都不影响，因均含旧机5,000账面） |
| 旧机处置价值 | — | (3,000) | 3,000 |
| 新机购置成本 | — | 12,500 | (12,500) |
| **成本合计** | 27,500 | 24,500 | **3,000（替换更优）** |

> 注意"处置损失 = 账面价值5,000 − 处置价值3,000 = 2,000"把**不相关的账面冲销**和**相关的处置价值**混在一起，所以**不要直接用处置损失**，要把两者分开列。

## 五、决策套路三：联产品——分离点出售 or 继续加工

> **联合成本(joint cost)在分离点前已发生，对"是否深加工"决策不相关！**

```
比较：继续加工后增加的收入(增量收入)  vs  分离点后单独发生的可分成本
若 增量收入 > 可分成本 → 继续加工；否则在分离点直接出售
```

### 例题 〔6-37〕
产品 M：分离点售价 30¢ vs 深加工成 Super M 售价 38¢，2,500,000 加仑：

| | 分离点卖M | 加工成Super M | 差异 |
|---|---|---|---|
| 收入 | 750,000 | 950,000 | +200,000 |
| 分离点后可分成本 | — | 210,000 | (210,000) |
| **结果** | 750,000 | 740,000 | **−10,000** |

→ **不应继续加工**（深加工反而少赚 10,000）。联合成本无论哪个方案都一样，**忽略不计**。

## 六、决策套路四：约束资源（Scarce/Limiting resource）

当某资源（机器工时、人工小时）不足时，应优先生产**单位约束资源贡献毛益最高**的产品：
```
排序依据 = 每单位约束资源的贡献毛益 = 单位贡献毛益 ÷ 单位产品耗用的约束资源
```
（注意：不是按单位贡献毛益排，而是按"每受限工时赚多少"排。）

## 七、单位成本陷阱（高频辨析）〔6-41〕

含固定成本的单位成本**只在某一产量下成立**，换产量必误导：
- 产量 10,000 件：(变动90,000 + 固定220,000) ÷ 10,000 = **$31/件**
- 产量 20,000 件：(变动180,000 + 固定220,000) ÷ 20,000 = **$20/件**
- 两个单位成本"同样准确（其实同样不准确）"。**结论：决策别直接乘单位成本，要拆成变动+固定分别处理。**

---

## 八、易错点与陷阱

1. **账面价值、账面冲销、已分摊的总部费用 = 沉没/不可避免 → 一律剔除。**
2. 旧设备的**处置价值**是相关的（现金流入）；别和账面价值混。
3. 自制外购时，只有**可避免**的固定成本才相关。
4. 联产品决策**忽略联合成本**，只比可分成本与增量收入。
5. 约束资源排序用"**每单位约束资源**的贡献毛益"，不是单位产品贡献毛益。
6. "超额折价/以旧换新"中，真实成本是**现金等价成本**（见 6-42：把"虚高折让"还原为标价的折扣）。
