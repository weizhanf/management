## 二、核心概念（中英对照）

| 概念 | 含义 |
|------|------|
| 变动成本 Variable cost | 总额随业务量**成正比变化**，单位变动成本不变（如原料、计件工资） |
| 固定成本 Fixed cost | 相关范围内**总额不变**，单位固定成本随产量增加而下降（如租金、月薪折旧） |
| 相关范围 Relevant range | 上述成本性态假设成立的业务量区间，超出则假设失效 |
| 贡献毛益 Contribution margin | 收入扣除全部变动成本后的余额，**先去补偿固定成本，剩余即利润** |
| 贡献毛益率 CM ratio | 单位贡献毛益 ÷ 单价 = 总贡献毛益 ÷ 销售收入 |

> **贡献毛益 ≠ 毛利(gross margin)**：贡献毛益是"收入 − 全部变动成本"；毛利是"收入 − 销货成本(COGS)"。
> 因为销货成本里既有变动也有固定，二者数值可能一大一小（见 2-53 例题）。

---

## 三、关键公式

```
单位贡献毛益 = 单价 P − 单位变动成本 V
总贡献毛益   = 销售收入 − 总变动成本 = 销量 Q × 单位贡献毛益
贡献毛益率   = 单位贡献毛益 ÷ 单价

营业利润 = 总贡献毛益 − 固定成本 F
        = Q × (P − V) − F

盈亏平衡点(销量) = 固定成本 F ÷ 单位贡献毛益 (P − V)
盈亏平衡点(销售额) = 固定成本 F ÷ 贡献毛益率

实现目标利润所需销量 = (固定成本 F + 目标利润) ÷ 单位贡献毛益
实现目标利润所需销售额 = (固定成本 F + 目标利润) ÷ 贡献毛益率

【含所得税】税前利润 = 税后目标利润 ÷ (1 − 税率)
           （再代入上面的"目标利润"公式）

安全边际 = 预计销售 − 盈亏平衡销售
安全边际率 = 安全边际 ÷ 预计销售
```

## 四、解题套路（贡献式利润表骨架）

所有第2章题目，先在草稿上画出这个"三层楼"，缺哪格用相邻格加减推出：

```
   销售收入        Sales
−  变动成本        Variable costs
─────────────────────────────
=  贡献毛益        Contribution margin   ← 关键中间量
−  固定成本        Fixed costs
─────────────────────────────
=  营业利润        Operating income
```

推导关系（任意缺口都能补）：
- 收入 = 变动 + 贡献毛益
- 贡献毛益 = 收入 − 变动
- 固定成本 = 贡献毛益 − 利润
- 利润 = 贡献毛益 − 固定成本

## 五、例题精讲（取自你的练习题）

### 例1 · 补全利润表 〔2-28〕
**情形1**（已知 收入900,000、变动500,000、固定350,000）：
- 贡献毛益 = 900,000 − 500,000 = **400,000**
- 净利 = 400,000 − 350,000 = **50,000**

**情形2**（已知 收入800,000、贡献毛益350,000、净利80,000）：
- 变动 = 800,000 − 350,000 = **450,000**
- 固定 = 350,000 − 80,000 = **270,000**

**情形3**（已知 变动600,000、贡献毛益340,000、固定250,000）：
- 收入 = 600,000 + 340,000 = **940,000**
- 净利 = 340,000 − 250,000 = **90,000**

> 要点：整张表就是上下相邻两格加减，**不需要任何额外公式**。

### 例2 · 用万能公式反解未知量 〔2-29〕
设：a=单价，b=单位变动成本，c=销量，d=总贡献毛益，e=固定成本，f=净利。
核心关系：`d = c × (a − b)`、`f = d − e`。

| 小题 | 已知 | 求解 | 答案 |
|------|------|------|------|
| 1 | a=25, c=120,000, d=720,000, e=640,000 | b, f | b=(d/c?)… 由 720,000=120,000(25−b) → **b=19**；f=720,000−640,000=**80,000** |
| 2 | a=10, b=6, c=100,000, e=320,000 | d, f | d=100,000(10−6)=**400,000**；f=400,000−320,000=**80,000** |
| 3 | a−b=5, d=100,000, f=15,000 | c, e | c=100,000÷5=**20,000**；e=d−f=100,000−15,000=**85,000** |
| 4 | a=30, b=20, c=70,000, f=12,000 | d, e | d=70,000(30−20)=**700,000**；e=700,000−12,000=**688,000** |
| 5 | c=80,000, d=160,000, e=110,000 | a, f | 160,000=80,000(a−9)→**a=11**；f=160,000−110,000=**50,000** |

### 例3 · 倒推目标收入 〔2-34〕
医院：变动成本率 30%，固定成本 42,000,000，要求**保本（利润=0）**：
```
设总收入 TR：TR − 0.30·TR − 42,000,000 = 0
0.70·TR = 42,000,000  →  TR = 60,000,000
```
（这里 0.70 就是贡献毛益率；保本收入 = 固定成本 ÷ 贡献毛益率 = 42,000,000 ÷ 0.70。）

### 例4 · 盈亏平衡 + 入住率 〔2-35〕
旅馆：每间 $50，单位变动成本 $10（→单位贡献毛益 $40），固定成本 3,200,000，全年可供 400×365=146,000 间。
```
保本间数 N：50N − 10N − 3,200,000 = 0
40N = 3,200,000  →  N = 80,000 间
入住率 = 80,000 ÷ 146,000 ≈ 54.8%
```

### 例5 · 多产品（电影+小卖部） 〔2-43〕
分产品列贡献毛益，再合计减固定成本：

| | 影片 | 小卖部 | 合计 |
|---|---|---|---|
| 收入 | 2,250 | 270 | 2,520 |
| 变动 | 1,125 | 162 | 1,287 |
| **贡献毛益** | 1,125 | 108 | **1,233** |
| 固定（场租330+人工435） | | | 765 |
| **营业利润** | | | **468** |

> 多产品题：**先分别算各产品贡献毛益 → 求和 → 再统一减固定成本。**

### 例6 · 贡献毛益 vs 毛利 + 敏感性 〔2-53〕（单位：百万）
- 贡献毛益 = 14,268 − 9,634 = 4,634；贡献毛益率 = 4,634 ÷ 14,268 = **32.5%**
- 若销售增长 10%：增量贡献毛益 = 14,268×10%×32.5% = **464**（固定成本不变）
- 这 464 全部转化为利润的改善。

## 六、易错点与陷阱

1. **固定成本不要"单位化"后乱用**：单位固定成本只在某一产量下成立，换产量就错（第6章 6-41 专门考这个）。
2. **贡献毛益率用"单价"做分母**，不是用成本。
3. **含税目标利润**：先把税后利润 ÷ (1−税率) 还原成税前，再代公式。
4. **多产品**必须分产品算贡献毛益，不能把固定成本摊到产品里再算。
5. **CVP 的前提假设**（简答常考）：成本能清晰分变动/固定、收入与成本线性、产量在相关范围内、效率与生产率不变、产品组合不变、存货变动可忽略。
