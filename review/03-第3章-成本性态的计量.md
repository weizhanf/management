# 第3章 · 成本性态的计量

> 英文：Measurement of Cost Behavior
> 本章解决"**怎么把一笔成本拆成固定部分和变动部分**"，为第2章 CVP 提供数据。

---

## 一、本章在考试中的地位

⭐⭐ **重点**。**高低点法（high-low method）是几乎固定出现的送分计算点**，务必练熟。其余多为概念（成本函数、约束性 vs 酌量性固定成本）。

---

## 二、核心概念（中英对照）

| 概念 | 含义 |
|------|------|
| 成本函数 Cost function | 成本与成本动因之间的数学关系，线性形式：`Y = a + bX` |
| —— a（截距） | 固定成本部分（总额） |
| —— b（斜率） | 单位变动成本（每单位成本动因） |
| —— X | 成本动因（业务量，如机器工时、产量） |
| 混合成本 Mixed cost | 同时含固定与变动两部分（如电话费=月租+通话费） |
| 阶梯成本 Step cost | 在某业务量区间内固定，跨过区间台阶式跳升 |
| 约束性固定成本 Committed FC | 由过去重大投资决定、短期难削减（厂房折旧、设备、关键人员薪酬） |
| 酌量性固定成本 Discretionary FC | 管理层每期可自主决定增减（广告、研发、培训、捐赠） |
| 产能成本 Capacity cost | 为提供一定生产/服务能力而发生的固定成本 |

> **约束性 vs 酌量性** 是高频概念辨析：判断标准是"管理层能否在短期内自主削减"。能削减→酌量性；不能→约束性。

---

## 三、计量成本性态的四种方法

| 方法 | 英文 | 做法 | 特点 |
|------|------|------|------|
| 工程分析法 | Engineering analysis | 按工艺、用料、工时逐项测算 | 精确但成本高，适合新业务 |
| 账户分析法 | Account analysis | 凭经验把每个账户判定为固定/变动/混合 | 简单快，主观性强 |
| 高低点法 | High-low method | 用最高、最低业务量两点求直线 | 计算简单，**只用两点、易受异常值影响** |
| 最小二乘回归法 | Least-squares regression | 用全部数据点统计拟合 | 最客观、最可靠，可给出 R² 拟合优度 |

---

## 四、解题套路：高低点法（必考）

> 注意：高低点选取依据是**成本动因(业务量 X)的最高与最低**，不是成本最高最低！

```
第1步  找出业务量 X 最高的那期 和 X 最低的那期（两期数据）
第2步  单位变动成本 b = (最高点成本 − 最低点成本) ÷ (最高点业务量 − 最低点业务量)
第3步  固定成本 a = 任一点总成本 − b × 该点业务量
第4步  写出成本函数 Y = a + bX，可用于预测其他业务量下的成本
```

### 示例
某月数据：最高业务量月 10,000 机器工时、总成本 $52,000；最低业务量月 6,000 机器工时、总成本 $40,000。
```
b = (52,000 − 40,000) ÷ (10,000 − 6,000) = 12,000 ÷ 4,000 = $3/机器工时
a = 52,000 − 3 × 10,000 = 52,000 − 30,000 = $22,000   （用最低点验算：40,000 − 3×6,000 = 22,000 ✓）
成本函数：Y = 22,000 + 3X
预测 8,000 工时的成本：Y = 22,000 + 3×8,000 = $46,000
```

---

## 五、关于回归分析（理解即可）

- 最小二乘法使"各点到拟合线的纵向距离平方和"最小，得到 a、b。
- **判定系数 R²**（0~1）衡量成本动因对成本变化的解释程度，越接近 1 拟合越好。
- 选择成本动因要兼顾两点：**经济合理性(plausibility)**（逻辑上确实驱动成本）和 **可靠性/拟合优度(reliability)**。

---

## 六、易错点与陷阱

1. **高低点的"高低"看业务量 X，不看成本**——这是最常见的扣分点。
2. 算固定成本 a 时记得用 `总成本 − b×业务量`，两点都可代入，可互相验算。
3. 约束性/酌量性固定成本区分：研发、广告是**酌量性**；厂房折旧、长期租约是**约束性**。
4. 高低点法**只用两个极端点**，所以容易被异常数据带偏；回归法更可靠。
