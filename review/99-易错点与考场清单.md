## 一、贯穿全书的"三大铁律"

1. **沉没成本永不相关**：旧设备账面价值、已花的研发费、已发生的任何成本，做决策时一律划掉。（第5、6、11章反复考）
2. **固定成本不要单位化后乱用**：单位固定成本只在某一产量成立，换产量必错。决策时把成本拆成"变动 + 固定"分别处理。（第2、6章）
3. **决策只看"未来 + 有差异"**：相关成本 = 未来才发生 且 各方案之间不同；机会成本必须计入。（第5、6章）

## 二、高频易错点清单（按章）

### 第2章 CVP
- [ ] 贡献毛益率分母用**单价**，不是成本
- [ ] 含税目标利润：先 `÷(1−税率)` 还原为税前
- [ ] 多产品先**分产品**算贡献毛益，再统一减固定成本
- [ ] 贡献毛益 ≠ 毛利（一个减全部变动成本，一个减销货成本）

### 第3章 成本计量（🔧只考概念，不手算）
- [ ] 约束性（厂房、折旧）vs 酌量性（广告、研发）别混 ← **概念高频**
- [ ] 会认成本函数 Y=a+bX 各符号；高低点法仅理解原理
- [ ] 题目直接给固定/变动成本时，会代入做 CVP/决策

### 第5章 定价决策
- [ ] 特殊订单**只算增量**，且需**有闲置产能**
- [ ] 成本加成率随"成本基础"不同而不同，看清用哪个
- [ ] 目标成本法是"售价倒推成本"，方向与成本加成相反

### 第6章 生产决策
- [ ] 自制外购只算**可避免**固定成本
- [ ] 旧设备**处置价值相关**、**账面价值不相关**，别混
- [ ] 联产品决策**忽略联合成本**
- [ ] 产品线增删：**可避免成本才相关**，不可避免分摊费用砍了也省不掉
- [ ] 约束资源按"每单位约束资源贡献毛益"排序

### 第7章 全面预算（🔧只考概念，不编表）
- [ ] 记得预算的 4 大作用、全面预算结构（选择/简答）
- [ ] 概念：现金预算不含折旧、编制起点是销售预算

### 第9章 责任会计（概念）
- [ ] 只对**可控项目**负责，剔除不可控分摊
- [ ] 平衡计分卡四维度能默写（财务/客户/内部流程/学习成长）
- [ ] 质量成本四类中**外部失败最贵**

### 第8章 弹性预算（概念）
- [ ] 静态预算 vs 弹性预算（弹性=调到实际产量）
- [ ] F=有利、U=不利；价格差异 vs 用量差异

> ⛔ 第10章（ROI/RI/EVA/转移价格）**不考**，不必复习。

### 第11章 资本预算
- [ ] **看清 Table 1（单笔）还是 Table 2（年金）**——头号失分点
- [ ] 折旧不是现金流，但**折旧×税率**抵税是现金流入
- [ ] 经营现金流要 `×(1−税率)`
- [ ] ARR 用**会计净利润**；有残值时平均投资=(初始+残值)/2
- [ ] 账面价值/沉没成本不相关
- [ ] NPV 与 IRR 结论一般一致；互斥方案优先看 NPV

## 三、计算题通用答题模板（考场照搬）

```
第1步  判题型：CVP / 决策 / 预算 / 资本预算？
第2步  画骨架：贡献式三层表，或"各方案并列"相关成本表
第3步  填数据：剔除沉没成本，保留未来差异，列出机会成本
第4步  下结论：写"应选 X 方案"，并补一句定性因素（质量/客户/风险）
```

> 即使最终数字算错，**列对表格结构、写对公式**也有步骤分。**绝不要只写一个孤零零的答案。**
